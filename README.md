# GACS-TCC

> **Geração de Aplicativo de Caracterização de Componentes Semicondutores**

## Sobre o projeto

O **GACS-TCC** é um projeto de Trabalho de Conclusão de Curso (TCC) do Bacharelado em Física da Universidade Estadual de Londrina (UEL).

O projeto tem como objetivo o desenvolvimento do **GACS – Gerenciador para Análise e Caracterização de Componentes Semicondutores**, um software destinado ao gerenciamento de experimentos, armazenamento de dados experimentais, análise e visualização de curvas de caracterização elétrica de componentes semicondutores.

Desde sua concepção, o sistema é desenvolvido com foco em simplicidade, organização, confiabilidade, extensibilidade e futura integração com equipamentos de laboratório.

O desenvolvimento é realizado de forma colaborativa entre o autor e o ChatGPT (OpenAI), utilizado como assistente de desenvolvimento para discussão de arquitetura, modelagem do sistema, implementação de código, documentação técnica, revisão de soluções e proposição de melhorias ao longo de todo o projeto.

---

## Primeira versão

A versão inicial do GACS será direcionada ao estudo e à caracterização elétrica de:

- Diodos;
- Transistores de Efeito de Campo (FETs).

Toda a arquitetura será projetada para permitir a futura inclusão de novos dispositivos semicondutores sem necessidade de reestruturação significativa do sistema.

---

## Objetivos

O GACS foi concebido para:

- gerenciar componentes semicondutores;
- cadastrar experimentos;
- armazenar dados experimentais em banco de dados;
- importar dados provenientes de planilhas;
- gerar gráficos de caracterização;
- comparar curvas experimentais;
- auxiliar na análise de resultados;
- servir como base para futura aquisição automática de dados laboratoriais.

## Roadmap de desenvolvimento

O desenvolvimento seguirá uma estratégia incremental:

1. Modelagem do domínio.
2. Estrutura do banco de dados.
3. Persistência.
4. Interface gráfica.
5. Importação e edição de dados.
6. Visualização de gráficos.
7. Ferramentas de análise.
8. Relatórios.
9. Integração com equipamentos laboratoriais.

## Filosofia do projeto

O GACS nasce com a proposta de contribuir para o desenvolvimento científico por meio de uma ferramenta aberta, organizada e extensível.
