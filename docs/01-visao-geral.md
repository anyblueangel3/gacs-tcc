# 01 -- Visão Geral do Projeto

# GACS-TCC

**Geração de Aplicativo de Caracterização de Componentes
Semicondutores**

## Finalidade

O GACS é um projeto de Trabalho de Conclusão de Curso (TCC) do
Bacharelado em Física da Universidade Estadual de Londrina (UEL).

Seu objetivo é desenvolver um software científico para apoiar
experimentos de caracterização elétrica de componentes semicondutores,
oferecendo uma plataforma única para entrada de dados, armazenamento,
visualização gráfica e caracterização.

## Motivação

Em muitos laboratórios os dados experimentais ficam dispersos entre
planilhas, arquivos e programas diferentes.

O GACS propõe integrar essas etapas em uma única aplicação, reduzindo
retrabalho, aumentando a organização dos experimentos e facilitando a
reprodutibilidade científica.

## Público-alvo

-   estudantes de Física;
-   pesquisadores;
-   laboratórios de caracterização de semicondutores;
-   desenvolvedores interessados em software científico.

## Objetivos do projeto

-   centralizar experimentos;
-   preservar dados experimentais;
-   gerar gráficos científicos;
-   apoiar a caracterização elétrica;
-   permitir evolução futura para aquisição automática de dados.

## Filosofia

O projeto prioriza:

-   simplicidade;
-   modularidade;
-   confiabilidade;
-   documentação;
-   evolução incremental.

O desenvolvimento é guiado por documentação arquitetônica antes da
implementação.

## Escopo da primeira versão

A primeira versão concentra-se em:

-   cadastro de experimentos;
-   entrada manual, colagem e importação de dados;
-   persistência;
-   geração de gráficos;
-   preparação para caracterização de diodos e FETs.

Itens avançados serão incorporados em versões posteriores.
