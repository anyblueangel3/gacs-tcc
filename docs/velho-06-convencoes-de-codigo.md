# 06 -- Convenções de Código

## Linguagem

Java 21.

## Organização

Pacotes:

-   application
-   controller
-   dao
-   model
-   service
-   util

## Domínio

Classes do domínio em português.

Exemplos:

-   Experimento
-   Coluna
-   DadoColuna
-   TipoEixo

## Convenções

-   Classes: PascalCase
-   Métodos e atributos: camelCase
-   IDs: Long
-   Datas: LocalDateTime
-   Valores medidos: Double (nesta fase)

## Comentários

Usar Javadoc em classes públicas e métodos cuja finalidade não seja
evidente.

## Objetivo

Priorizar legibilidade, simplicidade e facilidade de manutenção.
